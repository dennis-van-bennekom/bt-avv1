Browser Technologies

<picture>

De eerste HTML technologie waar ik een fallback voor wil maken is de <picture> element. Hiermee kun je voor bepaalde resoluties verschillende afbeeldingen tonen.

De fallback hierin is de <img> tag. Die wordt geladen als de <picture> niet ondersteund is.

caniuse

Can I Use picture? Data on support for the picture feature across the major browsers from caniuse.com.

Code

        
  <picture>
    <source
      media="(min-width: 800px)"
      srcset="images/720.jpg">
    <source
      media="(min-width: 500px)"
      srcset="images/1080.jpg">
    <img
      src="images/480.jpg"
      alt="Mooi bos">
  </picture>

<audio>

Het audio element wordt gebruikt om geluiden of muziek in een webpagina te laden.

De fallback hiervoor is een <a> element te gebruiken als de audio niet getoond kan worden, waardoor de gebruiker alsnog het bestand kan downloaden en beluisteren.

caniuse

Can I Use audio? Data on support for the audio feature across the major browsers from caniuse.com.

Code

        
  <audio controls="controls">
    <source src="audio/lekkernummer.mp3">
    <a href="audio/lekkernummer.mp3">Uw huidige browser ondersteund geen audio, download het bestand door hier te klikken.</a>
  </audio>

Viewport units

Viewports units kan je gebruiken om een percentage van je viewport te krijgen in je CSS.

De fallback hiervoor is om eerst een percentage of pixels te bepalen en daarna pas de viewport units.

caniuse

Can I Use viewport-units? Data on support for the viewport-units feature across the major browsers from caniuse.com.

Code

        
  .viewport {
    margin: 0 auto;
    height: 50px;
    width: 500px;
    width: 60vw;

    background-color: #DB5600;
  }

rem units

Rem units lijken op em, maar kijken naar de root element om hun grootte te bepalen. De em kijkt altijd naar de parent.

Een fallback hiervoor is het gebruik van een pixel waarde voordat je de rem waarde gebruikt.

caniuse

Can I Use rem? Data on support for the rem feature across the major browsers from caniuse.com.

Code

        
  .rem {
    font-size: 48px;
    font-size: 3rem;
  }

Geolocatie

Met geolocatie kan je toegang krijgen tot de locatie van het apparaat waar je website op draait.

Een fallback hiervoor is het tonen van een input veld als het ophalen van een locatie niet lukt.

caniuse

Can I Use geolocation? Data on support for the geolocation feature across the major browsers from caniuse.com.

Code

        
  <button id="geo">Locatie verkrijgen

  <div id="geo-fallback">
    <p>Locatie kon niet opgehaald worden. Voer hier uw locatie handmatig in:</p>
    <input type="text">
  </div>
        
      
        
  #geo {
    display: block;
    margin: 0 auto;
    padding: 0.5rem 1rem;
  }

  #geo-fallback {
    margin: 0 auto;
    max-width: 30rem;
  }

  #geo-fallback input {
    width: 100%;
  }
        
      
        
  (function(){
    var geoFallback = document.getElementById('geo-fallback');

    // Hide als geolocation er in
    geoFallback.classList.add('hidden');

    document.getElementById('geo').addEventListener('click', function() {
      if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(
          function(data) {
            alert("location: " + data.coords.latitude + ", " + data.coords.longitude);
          },
          function() {
            // Als het alsnog niet werkt, laat dan de fallback weer zien.
            geoFallback.classList.remove('hidden');
          }
        )
      } else {
        geoFallback.classList.remove('hidden');
      }
    });
  }());

Gamepad API

Met de gamepad API kan je controllers aansluiten en die gebruiken op het web.

Een fallback hiervoor zou kunnen zijn dat als er geen support of controllers zijn het toetsenbord gebruikt kan worden.

caniuse

Can I Use gamepad? Data on support for the gamepad feature across the major browsers from caniuse.com.

Code

        
  if (navigator.getGamepads) {
    window.addEventListener('gamepadconnected', function() {
      alert('Gamepad connected');
    });

    window.addEventListener('gamepaddisconnected', function() {
      alert('Gamepad disconnected');
    })
  } else {
    fallback();
  }

  function fallback() {
    // Use keyboard if gamepad is not supported

    alert('Gamepad not supported, please use your keyboard or update your browser');
    useKeyboard();
  }
